## 1) 핵심 User Flow

- 사용자가 중요한 의사결정 고민 등록 → 시스템이 고민을 저장 → 사용자는 고민 목록을 본다 → Admin에서 데이터가 확인됨
- 사용자가 헤더에서 조언가 역할로 전환 → 사용자가 등록한 고민 목록을 확인 → 해당 고민에 대한 조언을 등록 → 시스템이 조언을 고민에 연결하여 저장 → Admin에서 데이터가 확인됨
- Admin에서 고민과 연결된 조언을 승인 → 시스템이 조언을 조언과 연결된 고민 사용자에게 발송 + 조언 도착 알림 발송 → 사용자는 조언 목록을 본다 + 알림 확인 표시가 사라진다
