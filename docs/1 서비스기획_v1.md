# 서비스 도메인

- Keyword: 신뢰 · 사람 · 사람의 경험 · 이해 · 의사결정 · 결정 · 조언 · 인생 · 인생 설계 · 성공 · 시행착오 · 도메인 지식 · 암묵지 · 의사결정 도우미

# 문제

다양한 제한(환경: 이른 독립, 한부모 가정, 고아, 온라인 중심 등 / 내적: 성격 특성 등)에 따라 서민·중산층·주니어(청년에 국한하지 않는다)는 상위 계층 및 시니어 계층보다 절대적으로 인적 네트워크 풀과 경험 기반이 적다. 하지만 청년 시절에는 이러한 부재 자체를 쉽게 알아차리기 어려우며, 종종 잘못된 선택(의사결정)을 반복하게 된다.

## 본질적 문제

"나를 이해하는 현명한 어른"의 부재가 산업과 환경의 변화에 따라 줄어듦.
부모/삼촌/선배/교수/이웃/종교공동체 간의 교류(조언의 장)였던 레거시 네트워크가 필연적으로 줄어듦.
즉, 정보의 부족이 아닌, 수많은 정보 중 신뢰할 수 있고 압축하여 나에게 맞춰 잘 가공된 조언(사람)이 부족함.

# 해결 방안

서민·중산층·주니어에게 직관적이고 명확하게 사람(도메인 지식이 풍부한 시니어, 부자, 선배 창업가)의 이해와 가이드를 전달하는 플랫폼을 제공하고, 인생의 중요한 의사결정 과정에서 단일한 정답이 아닌 스스로 생각할 수 있는 과정을 제공한다.

# 기대 가치

정답이 아닌 스스로 생각할 수 있는 힘(과정)을 제공하여, 실패의 확률을 줄이는 것이 아닌,
성공하든 실패하든 '내가 직접 고민하고, 내가 직접 결정하여 실행했다'는 자기 주도적 가치와 실패 탄력성을
함유하게 함으로써 자아실현의 욕구를 주니어부터 빠르게 내재화할 수 있게 한다.

# 차별점

- AI가 아닌 신뢰할 수 있는 **사람의 조언**을 토대로 한다.
- AI는 보조 역할(기록·정리, 분석 보조, 매칭 브릿지, 신뢰 보증 장치)에 한정한다.
- 사람이 정답을 제시하지 않고, 스스로 생각하는 과정을 돕는다.

> 신뢰 점수 산정 로직, 조언자 평가 항목·기준, 승인 의사결정 트리 등 구체 설계는 비공개 부속 문서로 관리한다 (ADR-004).
